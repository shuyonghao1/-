力工一到店，所有喝酒的人便都看着他笑，有的叫道，“力工，你的老婆又拿你的钱去找健身教练了！”他不回答，对柜里说，“冰两瓶啤酒，要一碟炒青菜。”便拿出结婚证拍在桌上。他们又故意的高声嚷道，“你一定又倾家荡产花三十八万八当冤大头了！”力工睁大眼睛说，“你怎么这样凭空污人清白。”“什么清白？我昨天看见你的老父亲到处求亲戚低三下四筹到钱亲手送到女方家里，差两千块被人赶了出去。”力工便涨红了脸，额上的青筋条条绽出，争辩道，“娶老婆不能算冤大头，娶老婆！传宗接代的事，能算冤大头吗？”接连便是难懂的话，什么“男人赚钱养家，女人貌美如花”，什么“她不一样”之类，引得众人都哄笑起来，店内外充满了快活的空气。 收起
